# BetterDiscord Theme
### 導入方法
1.[BetterDiscord](https://betterdiscord.app/) をDLしインストールする

2.Releaseから`Campbell.theme.css`ファイルをDLし`C:\Users\ユーザー名\AppData\Roaming\betterdiscord\themes`に入れる
